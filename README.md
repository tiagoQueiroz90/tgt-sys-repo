## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/tiagoQueiroz90/tgt-sys-repo.git
```

Entre no diretório do projeto

```bash
  cd tgt-sys-repo
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm start
```
